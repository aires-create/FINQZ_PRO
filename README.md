# 🚀 FINQZ_PRO

**Sistema FINQZ** — Plataforma de automação e gestão financeira para negócios modernos.

> Automatize processos, gerencie clientes e tenha controle total da sua operação financeira em um só lugar.

---

## 🎯 Visão do Produto

O **FINQZ** nasce com o objetivo de simplificar e automatizar a gestão financeira de empresas, integrando:

* 📊 Controle financeiro
* 🤖 Automação de processos
* 👥 Gestão de clientes (CRM)
* 📈 Insights e relatórios

## 🧱 Arquitetura do Projeto

```bash
/backend   # API, lógica de negócio e automações
/src       # Frontend da aplicação
```

---

## 🛠️ Tech Stack

* **Frontend:** Vite + TypeScript + TailwindCSS
* **Backend:** Node.js (estrutura modular)
* **Build & Tools:** PNPM, ESLint, Vitest

---

## 🚀 Como rodar o projeto

```bash
# Instalar dependências
pnpm install

# Rodar projeto
pnpm dev
```

> Certifique-se de configurar o arquivo `.env` baseado no `.env.example`

---

## 🔐 Variáveis de Ambiente

Crie um arquivo `.env` baseado em:

```bash
.env.example
```

E configure conforme necessário para seu ambiente.
